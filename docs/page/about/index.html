About

This is a blog mostly dedicated to the fml project, a C++ framework with R bindings for high-performance dense matrix computing. Sometimes I also talk about math/computing things that I’m thinking about, usually in relation to fml.

I use a lot of LaTeX in posts, so enabling JavaScript for MathJax will make things look much nicer. I don’t use any kind of trackers (that I’m aware of). Currently I use GitHub Pages for hosting. The blog framework is Hugo with a slightly modified Hugo theme onetwothree. Source for the blog is available here.

Posts are copyright CC-BY-SA. Source code listings within a post are CC0.

If you find an errors or enjoyed something, please let me know. I go by wrathematics on twitter and gmail.
