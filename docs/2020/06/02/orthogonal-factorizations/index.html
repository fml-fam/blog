Orthogonal Factorizations
/math   

Written Tue, June 2, 2020.

Integers can be factored into products of special kinds of integers with useful properties called primes. Similarly, matrices be factored into special kinds of matrices with useful properties.

Anyone who took a matrix algebra course in university has seen the LU decomposition. This is useful for things like solving square systems of equations and computing matrix inverses. Probably the most important/powerful of the factorizations is the singular value decomposition.

QR factorization.

QR algorithm

Factor $A=QR$, where $R$ is upper triangular, $Q^TQ = I$
